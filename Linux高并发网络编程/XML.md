# xml

1 有一个文件头

```xml
<?xml version="1.0" encoding="ut-8"?>
```

2 由一系列的标签组成: 逻辑上是树结构

- 根标签只有一个

- 一般成对出现

  ```
  <hello></hello>
  ```

- 不成对: 

  ```
  <hello/>
  ```

  

3 标签区分大小写

4 标签可以设置属性

```
<font color="red"></font>
```

5 注释

```
<!-- xxxx -->
```
